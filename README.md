# Form Submission Project (GET & POST)

##  Features
- HTML Form using GET & POST methods
- Backend using Node.js & Express
- MySQL database integration
- Data stored securely in database
- Success popup after submission

## 🛠Tech Stack
- HTML, CSS
- Node.js, Express.js
- MySQL

##  How to Run

### Backend
cd Backend
npm install
node server.js

### Frontend
Open index.html using Live Server

##  Learnings
- Difference between GET and POST
- Handling form data in backend
- Connecting Node.js with MySQL
